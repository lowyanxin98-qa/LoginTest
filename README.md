# Selenium Login Automation Framework

## 📌 Overview
This project demonstrates automated testing of a login functionality using Selenium WebDriver and TestNG.  
It follows the Page Object Model (POM) design pattern for better maintainability and readability.

---

## ⚙️ Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

---

## 🧱 Project Structure
- **BaseTest**: Handles browser setup and teardown  
- **Pages**: Page Object Model classes (LoginPage, ProductsPage)  
- **Tests**: Test cases using TestNG  

---

## 🧪 Test Scenarios
- ✅ Valid login test  
- ❌ Invalid username test  
- ❌ Invalid password test  

---

## 🚀 How to Run
1. Clone the repository  
2. Open in IntelliJ  
3. Run `testng.xml` or test classes directly  

---

## 📊 Notes
- Uses explicit waits for better stability  
- Focuses on clean structure and reusable components
