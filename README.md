# Selenium Login Automation Framework

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

## Project Structure
- BaseTest: handles browser setup and teardown
- Pages: Page Object Model classes
- Tests: Test cases using TestNG

## Test Scenarios
- Valid login test
- Invalid username test
- Invalid password test

## How to Run
1. Clone the repository
2. Open in IntelliJ
3. Run testng.xml or test classes directly
